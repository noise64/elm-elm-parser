elm-elm-parser
==============

This is a playground repo with the aim of creating an elm parser in elm. 

Things to install before build
------------------------------
 - java
 - node / npm (http://nodejs.org/)
 - elm (http://elm-lang.org/)

How to build, run, test?
------------------------
```sh
# build
./gradlew build

# run
./gradlew run

# test
./gradlew test
```
